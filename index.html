Bio-Edu Suite

ウェットな分子生物学実験から、最先端のバイオインフォマティクスまでをシームレスに繋ぐ、教育用インシリコ・プラットフォーム。

Version 1.1
Phase 1 ウェット実験の設計と準備 (Wet Lab Prep)

① Master Mix Studio

PCR反応に必要なDNAの希釈率と、マスターミックスの分注量を自動計算します。

👨‍🏫 なぜ学ぶのか？ピペッティング誤差を防ぐ「N+1」の概念や、粗抽出液の阻害物質を薄める根拠など「プロの実験作法」を身につけます。
アプリを起動 🚀

② Thermal Cycler Simulator

酵素の特性やターゲット長、プライマーのTm値に基づき、最適なPCRプログラムを構築します。

👨‍🏫 なぜ学ぶのか？サーマルサイクラーの設定は勘ではありません。生物学的・物理的根拠に基づいて機械を制御する論理的思考を養います。
アプリを起動 🚀
Phase 2 仮想実験による検証 (In Silico Validation)

③ Virtual PCR & RFLP

FASTAデータを読み込み、PCR増幅と制限酵素による切断（RFLP）をシミュレートし、バーチャルゲルに描画します。

👨‍🏫 なぜ学ぶのか？プライマーのミスマッチ許容や、制限酵素が「どこを・なぜ切るか」を視覚化し、対数スケールの実際の電気泳動を読む目を養います。
アプリを起動 🚀
Phase 3 波形解析と配列の整形 (Sequence Curation)

④ DNA Alignment Studio

FASTAを整列させ、ノイズ除去、簡易MSA（多重整列）、そして新しいプライマーの設計（Tm計算）を行います。

👨‍🏫 なぜ学ぶのか？DNAの共通領域や特異的な変異（SNP）を見つけ出し、「ユニバーサル」または「種特異的」なプライマーを自らデザインする視点を持ちます。
アプリを起動 🚀

⑤ Alignment Print Studio

アライメント結果（CLUSTAL形式）を、ページまたぎによる分断を防ぎながら、論文・レポート用に美しくPDF出力します。

👨‍🏫 なぜ学ぶのか？等幅フォントの使用や、配列ブロックの分断防止など、科学論文において厳格に定められた「データ提示の世界標準ルール」を守らせます。
アプリを起動 🚀

⑥ Sanger Trace Editor

シーケンサーの生データ（.ab1）を解読して波形を表示し、手動で塩基の修正を行いFASTAを出力します。

👨‍🏫 なぜ学ぶのか？機械の自動判定を盲信せず、波形の乱れやノイズを人間の目で確認して品質を担保する「バイオインフォマティクスの第一歩」を体験します。
アプリを起動 🚀
Phase 4 高度な情報解析の裏側 (Advanced Bioinformatics)

⑦ Virtual BLAST Explorer

未知の配列とデータベースを比較し、種同定のための「スコアリング」と「E-value」の計算を体験します。

👨‍🏫 なぜ学ぶのか？BLASTをブラックボックスにせず、「E-value（偶然一致する確率）」の重要性や、配列が短すぎる際のバーコーディングの限界を学びます。
アプリを起動 🚀

⑧ Phylogeny Builder

アライメントから「距離行列」を作成し、UPGMA法やNJ法で系統樹の枝を1つずつ結合していく過程をエミュレートします。

👨‍🏫 なぜ学ぶのか？MEGAなどのソフトが裏側で行っている「一番似ているもの同士をくっつけて表を更新する」という計算プロセスを視覚的に理解します。
アプリを起動 🚀
Phase 5 タンパク質の立体構造と機能 (Structural Biology)

⑨ Protein Structure Explorer

PDBデータやAlphaFold予測を用いてタンパク質の3D構造を観察し、DNAの変異が立体構造に与える影響を視覚化します。

👨‍🏫 なぜ学ぶのか？「配列の変異」が「構造と機能の異常」に直結する様子を3Dで体感し、AI予測と実験値の違いを見極める構造リテラシーを養います。
アプリを起動 🚀
