🧬 Bio-Edu Suite

Bioinformatics & Wet-Lab Education Platform

Bio-Edu Suiteは、DNA配列の増幅からタンパク質の立体構造解析まで、
バイオサイエンスのパイプラインをブラウザ上で直感的に学べる教育用アプリケーション群です。
Phase 1 遺伝子増幅と確認 (PCR & RFLP)

① PCR Master Mix Studio

実験の第一歩。Taqポリメラーゼやプライマーなど、PCRに必要な試薬の最適な混合比率（マスターミックス）を計算します。

👨‍🏫 なぜ学ぶのか？酵素の入れすぎによる阻害など、現場でよく起きる「手技と濃度のミス」を事前に防ぐ感覚を養います。
アプリを起動 🚀

② Thermal Cycler Simulator

サーマルサイクラーの温度と時間を設定し、DNAが変性・アニーリング・伸長していく過程をシミュレートします。

👨‍🏫 なぜ学ぶのか？「なぜこの温度にするのか？」という熱力学的な根拠（Tm値など）を理解し、トラブルシューティング力を鍛えます。
アプリを起動 🚀

③ Virtual PCR & RFLP

仮想のPCR増幅と制限酵素による消化を行い、アガロースゲル電気泳動のバンドパターンを予測・確認します。

👨‍🏫 なぜ学ぶのか？1塩基の変異（SNP）が制限酵素の認識部位を変化させ、バンドの数と長さを変える原理（RFLP法）を視覚的に理解します。
アプリを起動 🚀
Phase 2 シーケンス解析とアライメント (Sequence Analysis)

④ Sanger Trace Editor

サンガーシーケンシングの生データ（.ab1波形ファイル）を読み込み、波形の乱れ（ノイズ）を目視で確認して手動で修正・カットします。

👨‍🏫 なぜ学ぶのか？機械が出したA/T/G/Cの文字列を鵜呑みにせず、生の波形データに戻って真偽を確かめる「研究者の目」を養います。
アプリを起動 🚀

⑤ DNA Alignment Studio

【パイプライン・ハブ】 生配列のノイズ領域をプライマー検索で特定・カットし、複数配列のアライメント（位置合わせ）を行います。

👨‍🏫 なぜ学ぶのか？解析前にデータの開始位置を揃える（キュレーションする）ことの重要性と、アプリ⑨へデータを渡すパイプラインの概念を学びます。
アプリを起動 🚀

⑥ Alignment Print Studio

アプリ⑤で作成したアライメント結果（Clustal形式）を読み込み、論文やレポートに添付できる美しいPDF形式で出力します。

👨‍🏫 なぜ学ぶのか？解析結果を「他人に分かりやすく共有する」という、研究発表において不可欠なプレゼンテーション技術をサポートします。
アプリを起動 🚀
Phase 3 データベース検索と系統分類 (Bioinformatics)

⑦ Virtual BLAST Explorer

未知のDNA配列を入力し、データベースから類似する配列を検索する「BLAST」のアルゴリズムをシミュレートします。

👨‍🏫 なぜ学ぶのか？スコアやE-valueの意味を理解し、手元の配列がどの生物の何の遺伝子であるかを特定するバイオインフォマティクスの基本を学びます。
アプリを起動 🚀

⑧ Phylogenetic Tree Builder

アライメント結果から距離行列を作成し、UPGMA法やNJ法を用いて生物間の進化的な近さを示す「系統樹」を構築します。

👨‍🏫 なぜ学ぶのか？「配列が似ている＝進化的に近い」という概念を、計算アルゴリズムの視覚化を通じて直感的に理解します。
アプリを起動 🚀
Phase 4 セントラルドグマと構造生物学 (Central Dogma & Structure)

⑨ Central Dogma Simulator

DNAからmRNAへの転写、そしてタンパク質への翻訳プロセスをシミュレートし、ミスセンス変異によるアミノ酸の性質変化を検出します。

👨‍🏫 なぜ学ぶのか？「DNAの変異」が「アミノ酸の性質変化」を引き起こすという、生命の基本原理（セントラルドグマ）を実データで追体験します。
アプリを起動 🚀

⑩ Protein Structure Explorer

PDBデータを用いてタンパク質の3D構造を観察し、アプリ⑨で検出した変異が立体構造のどこに位置するかを視覚化します。

👨‍🏫 なぜ学ぶのか？「アミノ酸の性質変化」が最終的に「タンパク質の機能異常（立体構造の崩壊）」に繋がる因果関係を3D空間で推論します。
アプリを起動 🚀
