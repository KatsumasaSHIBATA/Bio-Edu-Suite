【根拠：なぜVOLを正しく設定するのか？】
PCR機器は、入力された「VOL」の液量に基づいて、ヒーターの加熱・冷却パワー（熱容量）を自動計算しています。ここが間違っていると、設定温度に達するスピードが狂い、実験が失敗します。

【表示が小数点第2位までの理由】
アプリ1から読み込んだ「総液量」に基づいて比率（スケール）計算が自動で行われるため、割り切れない細かい数値が出ることがあります。ここでは「計算上の正確な必要量」を可視化するためにあえて小数点第2位まで表示しています。（実際の実験では、これを四捨五入してピペットの目盛りに合わせて測ります）

📖 クイックガイド (ThermoGen QuickBath準拠)

※本シミュレーターは、実際のPCR機器のマニュアル記述と操作感を完全に再現しています。

【画面表示の用語】
SV = 設定温度 (°)
TM = 温度の保持時間 (分 : 秒)
PV = 運転時のサンプル液相当温度 (°)
N= = サイクルの繰り返し回数

■ 運転の準備 (HOME / SETUP)

1. 起動時は左上の "Run-ProgName" にカーソル（■）があります。
2. [▼] [▲] キー または [SEL] キーで "Template" の前にカーソルを移動し、[ENT] キーを押します。
3. "Edit" を選択して運転条件画面 (SETUP) を表示させ、サンプル量（VOL）等を入力します。
4. [→] キーを押してパート1の画面に進みます。
⚠️ やけど注意：カバーの内面は高温になります。触らないでください。

■ プログラムの入力 (PART 1〜3)

1. [→] [←] キーで、画面（パート）を切り替えます。
2. [▼] [▲] キーで設定したい項目の前にカーソルを移動させます（選択項目はマーカー ↓ や ■ が点滅）。
3. [ENT] キーを押すと編集モードに入ります。
4. テンキーで数値を入力します。
　・例：94.5°の場合、「9」「4」「5」
　・例：2分30秒の場合、「0」「2」「3」「0」
　💡 無限大(∞)の入力：PART3の最終時間で「9」「9」「9」「9」と入力します。
5. 再度 [ENT] キーを押して確定します。
❗ こんなときは････
数字を訂正する場合、[C] キーを押すと入力中の数字がクリアされ、先頭からやり直せます。

■ プログラムの保存と呼出 (STORE)

1. 保存： PART1・3 画面で [SEL] キーを押し "Store" にカーソルを合わせて [ENT] キーを押します。
2. 呼出： HOME 画面から "StoredPrg" を選択します。
3. ページ切替： 保存庫画面で [→] [←] キーを押すとページが切り替わり、最大18個保存できます。
4. 場所の選択： [SEL] キーで任意の場所（空きは空欄）を選び、[ENT] キーを押します。
5. 名前入力： [↑] [↓] キーでアルファベットを選び、テンキーで数字を入力。[→] キーで次の文字へ進み、最後に [ENT] キーで確定します。（上書きの確認が出た場合は [SEL] で Yes を選び [ENT]）

■ 運転スタート・カバー加熱・停止

1. スタート： "RUN" にカーソルを合わせて [ENT] キーを押すと "Standby?" 画面が出ます。
　・Yes： 待機状態(PAUSE)になります。サンプルをセット後、[PAUSE/RUN] キーを押すと開始します。
　・No： すぐにカバーの加熱 (Lid is heating) が始まります。
2. カバー加熱： 待機中はパネルの LID HEATランプが赤く点滅 します。（[→] キーでスキップ可能）完了するとランプが点灯し、PCRが開始されます。
3. 運転中： PVの横に ＞ が表示され、現在の温度の ° が点滅します。
4. 一時停止 / 停止： 青色の [PAUSE/RUN] キーで一時停止・再開ができます。完全停止は赤色の [STOP] キーを押し、確認画面で [SEL] で "Yes" を選び [ENT] を押します。

※1: 初期変性(INIT)はDENATと同じ（または94℃以上）に設定し、ホットスタート酵素の場合は約2分間加熱して活性化させます。
※2: 最終伸長(FINAL)はEXTENDと同じ至適温度に設定し、未完成のDNAを最後まで合成させます(通常1〜5分)。
