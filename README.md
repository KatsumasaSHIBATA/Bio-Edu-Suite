# 🧬 Bio-Edu Suite (バイオ教育スイート)

## 🎯 概要 (Overview)
Bio-Edu Suiteは、大学や高校の生物学・バイオインフォマティクス実習のために開発された、**ブラウザだけで完結する教育用ウェブアプリケーション群**です。
学生が「1Dの遺伝子配列」から「3Dのタンパク質立体構造」までの繋がりや、PCRの原理などを、手を動かしながらインタラクティブに「腹落ち」して学ぶことができます。

## ✨ 主な特徴 (Features)
* **完全ブラウザ完結 (Zero Setup):** サーバーやPython環境などの面倒な構築は一切不要。学生のPCやスマホで、HTMLファイルを開くだけで即座に動作します。
* **教育的UI設計:** 単なる解析ツールではなく、「なぜその温度にするのか？」「なぜその分子を見るのか？」という現場の根拠やコツを解説するパネルを常備しています。
* **1D-3D 究極の双方向連動:** アミノ酸配列(1D)の文字をクリックすると、立体構造(3D)の該当箇所がリアルタイムにズーム・強調されるなど、直感的な操作性を実現しました。
* **ドラッグ＆ドロップ対応:** FASTA形式の配列データや、PDB/CIFの構造データを画面に投げ込むだけでシームレスに解析が始まります。

## 📦 収録アプリケーション
現在、以下の主要な教育用シミュレーターが収録されています。

* **`2_Thermal_Cycler_Simulator.html`**
  * PCRのプロトコル最適化と論理的トラブルシューティングを学ぶシミュレーター。
* **`4_DNA_Alignment_Studio.html`**
  * 複数のDNA配列を比較し、変異箇所を特定するアライメントエディタ。
* **`9_Protein_Structure_Explorer.html`**
  * 1D-3D連動型 タンパク質立体構造ビューア。PDBの実測データや、AlphaFoldのAI予測データ（ローカルファイル）の描画に対応。

## 🚀 使い方 (How to Use)
1. 本リポジトリの右上にある緑色の `[<> Code]` ボタンから `Download ZIP` を選び、ファイル一式をダウンロードして解凍します。
2. 使いたいアプリの `.html` ファイルをダブルクリックし、Google Chrome や Safari などのブラウザで開きます。
3. 画面の指示に従い、テキスト入力やファイルのドラッグ＆ドロップを行って実習を進めてください。
